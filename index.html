AutoRelaunch

written in: C#

This is my first serious project related to programming. This is a troll app that, as the name suggests, will relaunch a selected application whenever it's closed.

How to use:

1. Launch the app
2. Point to the exe file of the application
3. Input the name of the process (without the extension)
4. (Optional) Change the delay
Now click "Activate" and "Hide this window"

The only way to stop it now is through task manager.

Project link:

github.com/counter185/autoRelaunch

EmojiStudio

written in: C#

This is my first big finished Android app. Rather than using Android Studio, i used Xamarin.
This is an app that allows you to create emoji art. It was originally a port of a python script I made that could convert BMP files to emoji art.
The app was supposed to be easy to use, so instead of converting files, I decided to switch to making an entire editor.
I originally tried to upload this onto Amazon's app store, but it was rejected.

How to use:

1. Launch app
2. Color code your emoji art
3. Change the color coding
4. Copy to clipboard
5. Disappoint your WhatsApp family chat

Project link:

github.com/counter185/emojiStudio
